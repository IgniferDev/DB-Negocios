# Proyecto-Final — Sistema de Soporte de Decisión (DSS) - BUAP

Estructura y scripts para el proyecto final: ETL, DW, OLAP web y modelo predictivo (Rayleigh).

## Requisitos
- Python 3.8+
- SQL Server (Ponki: PROJECT_MANAGE, Kiry: PROJECT_SUPPORT_SYSTEM)
- ZeroTier configured for network accessibility (optional)
- Instalar dependencias:
""pip install -r requirements.txt""
